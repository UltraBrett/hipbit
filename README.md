hipbit
======
Created by Thomas Turney

Hipbit Service for html5 remote music streaming

Building:
============
  Requires python 2.7.x
  wxpython-2.8
  pyinstaller-2.1
  ID3-lib-1.2

Building:
  
Unit testing: pyinstaller -F -w wxgtk

User setup program: pyinstaller -F -w Bitblocks.py --icon="folder_music.ico"
